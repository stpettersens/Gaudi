Gaudi depends on the following...

Tools & Runtimes

(Necessary to build from source without using Gaudi itself
or run Gaudi itself)
  • Python
    (python 2.7+ - recommended: CPython or PyPy)
  • Apache Ant
    (apache ant 1.8+)
  • Scala
    (scala distribution 2.9.0.1)
  • Java Runtime Environment (JRE)
    (recommended: HotSpot JRE, OpenJDK JRE or GNU GIJ)
  • Java Development Kit (JDK)
    (recommended: Oracle JDK or OpenJDK JDK or GNU GCJ)

Libraries

(Necessary to both build and run Gaudi)
  • JSON.simple
    (json_simple-1.1.jar)
  • Apache Commons IO
    (commons-io-2.0.1.jar)
  • Scala library
    (scala-library.jar 2.9.0.1 final)
  • Groovy library
    (groovy-all-1.8.0.jar)
  • Jython (minimal core library)
    (jython.jar 2.5.2)
  • java-gnome
    (gtk-4.0.jar & native library)
